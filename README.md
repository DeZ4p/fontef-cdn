## 📁 Files

| File | Purpose |
|------|---------|
| `ui.html` | 🖥️ Complete WebUI layout |
| `style.css` | 🎨 WebUI stylesheet with dark/light theme |
| `app.js` | ⚙️ WebUI logic and universal KSU bridge |
| `config.json` | 📋 Central configuration (languages, presets, theme) |
| `update.json` | 🔄 Module update manifest |
| `changelog.md` | 📜 Version changelog |

## 🎛️ Configuration via config.json

| Section | Controls |
|---------|----------|
| `branding` | 🏷️ App name, logo, footer, badges |
| `languages` | 🌍 UI languages and translations |
| `presets` | 📱 ROM preset buttons |
| `previews` | 👁️ Font preview texts |
| `font_targets` | 📄 System font files to replace |
| `theme` | 🎨 Color scheme |
| `ui` | ⚙️ Component visibility |

## ⚡ Advantages

- 🚀 Instant UI updates without reinstalling module
- 🌍 Add languages remotely
- 🎨 Change theme from config
- 📱 Works on ALL KernelSU devices
- 💰 Free hosting (GitHub Pages)
- 🔒 Read-only for users
